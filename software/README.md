# Getting Started

The MSPM0C1104SDSGR Devlab is a compact development board featuring the microcontroller ARM Cortex-M0+ MSPM0C1104SDSGR. This board is designed for developers and hobbyists to prototype and test applications with ease.

## Prerequisites

Before you begin, ensure you have the following:

- A computer with a USB port
- The latest version of [Code Composer Studio](https://www.ti.com/tool/CCSTUDIO) or [IAR Embedded Workbench](https://www.iar.com/iar-embedded-workbench/)
- Basic knowledge of embedded systems and programming

## Getting Started Steps

1. Connect the MSPM0C1104SDSGR Devlab to your computer using a USB cable.
2. Install the necessary drivers for the development board.
3. Open your preferred IDE (Code Composer Studio or IAR Embedded Workbench).
4. Create a new project and select the MSPM0C1104SDSGR as the target device.
5. Write your application code and build the project.
6. Flash the compiled firmware to the Devlab using the built-in programmer.
7. Start prototyping and testing your applications!
8. Refer to the [hardware documentation](./hardware/README.md) for detailed information on pin configurations and functionalities.
